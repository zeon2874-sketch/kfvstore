# 프로젝트명
한국민속촌 스토어 월간 성과 streamlit 대시보드 제작

# 목적
아래 raw 데이터 파일들을 기반으로, 월별 성과를 한눈에 확인할 수 있는 streamlit 대시보드를 제작한다.
대시보드는 경영진 및 실무자가 함께 볼 수 있도록 직관적이고, KPI 중심으로 구성한다.
사용자가 기간을 직접 선택하여 월별 성과를 조회할 수 있어야 한다.

# 사용 데이터
1. top3 매출 업체
- 월별 TOP3 매출 업체 데이터

2. 목표 달성률
- 월간 매출 목표
- 월간 회원 목표
- 월별 목표 대비 실적 및 달성률 계산용 데이터

3. 브랜드사업본부_월간결산_루커
- ERP 기준 실제 구매확정 건에 대한 월별 매출 정리본
- 포함 항목: 매출, 매출원가, 영업이익, 당기순이익, 판관비, 마케팅비, MAU 등

4. 정식몰 주문집계(9_1~)
- 환불/취소건 반영되지 않은 주문 집계 파일
- 상품/카테고리/업체/주문 기준 분석용

5. 정식몰 회원집계(9_1~)
- 회원수 집계 파일
- 월별 회원 KPI 확인용

# 데이터 사용 원칙
- 매출/이익/비용/마케팅비/MAU 등 경영지표는 반드시 "브랜드사업본부_월간결산_루커" 파일을 우선 기준으로 사용한다.
- 상품 판매 분석, 카테고리 분석, 상위 업체 분석, 많이 팔린 상품 분석은 "정식몰 주문집계(9_1~)" 파일을 사용한다.
- 회원 관련 수치는 "정식몰 회원집계(9_1~)" 파일과 "목표 달성률" 파일을 활용한다.
- 동일한 지표가 여러 파일에 존재할 경우, 아래 우선순위를 따른다.
  1순위: 브랜드사업본부_월간결산_루커
  2순위: 목표 달성률
  3순위: 정식몰 주문집계(9_1~)
  4순위: 정식몰 회원집계(9_1~)
- "정식몰 주문집계(9_1~)"는 환불/취소가 반영되지 않은 파일이므로, 총매출 KPI 용도보다는 상품/브랜드/카테고리 트렌드 분석 용도로 활용한다.

# 대시보드 필수 포함 항목
아래 항목은 반드시 구현한다.

## 1. 매출 및 회원 목표 달성률
- 월별 매출 목표
- 월별 실제 매출
- 매출 목표 달성률(%)
- 월별 회원 목표
- 월별 실제 회원 수
- 회원 목표 달성률(%)
- 카드 또는 게이지 차트로 직관적으로 표현
- 목표 대비 초과/미달 여부도 한눈에 보이도록 표시

## 2. 상품군 판매 분석(카테고리 분석)
- 월별 카테고리별 매출
- 월별 카테고리별 판매수량
- 카테고리 비중 분석
- 카테고리별 순위 확인 가능하도록 구성
- 필요 시 막대차트, 트리맵, 도넛차트 활용

## 3. 상위 3개 매출 업체 및 매출액
- 선택한 월 기준 상위 3개 업체 추출
- 업체명, 매출액, 판매수량 표시
- 전월 대비 증감이 가능하면 함께 표시
- 상위 3개 업체가 한눈에 비교되도록 가로 막대차트 또는 랭킹 카드로 구성

## 4. 가장 많이 팔린 상품 TOP5
- 선택 월 기준 판매수량 TOP5 상품
- 상품명, 브랜드/업체명, 판매수량, 매출액 표시
- 상품별 매출 기여도 확인 가능하게 구성

## 5. 마케팅비 + 매출대비 ROAS
- 월별 마케팅비
- 월별 매출
- ROAS 계산
- ROAS = 매출 / 마케팅비
- 마케팅비와 매출을 함께 비교할 수 있는 차트 구성
- ROAS는 별도 KPI 카드로 강조

## 6. 영업이익
- 월별 영업이익 KPI 카드
- 가능하면 전월 대비 증감률도 함께 표시

## 7. 매출
- 월별 총매출 KPI 카드
- 가능하면 전월 대비 증감률 표시

## 8. 회원
- 월별 회원 수 KPI 카드
- 신규회원 중심이면 신규회원 수로 명확히 표기
- 누적회원과 월간회원이 구분 가능하면 함께 표시

## 9. MAU
- 월별 MAU KPI 카드
- 추이 확인 가능하도록 시계열 차트 포함

# 필터/인터랙션 요구사항
- 사용자가 기간을 직접 선택할 수 있어야 한다.
- 기본 단위는 "월별 조회"로 한다.
- 월 선택 필터를 반드시 제공한다.
- 가능하면 기간 비교 기능도 고려한다.
  - 예: 전월 대비
  - 예: 특정 월 vs 특정 월
- 모든 시각화는 선택한 월 기준으로 동기화되도록 한다.

# 추천 대시보드 구조
대시보드는 1페이지 또는 2페이지 이내로 구성한다.

## [페이지 1] 경영 KPI 요약
상단 KPI 카드:
- 매출
- 영업이익
- 회원
- MAU
- 마케팅비
- ROAS
- 매출 목표 달성률
- 회원 목표 달성률

중단:
- 월별 매출/회원 목표 대비 실적 차트
- 월별 매출, 영업이익, 마케팅비 추이 차트
- 월별 MAU 추이 차트

하단:
- 핵심 인사이트 영역
- 예: "이번 달 매출은 목표 대비 132% 달성"
- 예: "회원은 목표 대비 미달, MAU는 증가"
- 예: "ROAS는 전월 대비 개선"

## [페이지 2] 상품/업체 분석
상단:
- 카테고리별 매출 및 판매수량 분석
- 카테고리 비중 차트

중단:
- 상위 3개 매출 업체 랭킹
- 업체별 매출액 비교 차트

하단:
- 가장 많이 팔린 상품 TOP5 테이블 또는 랭킹 차트

# 데이터 전처리 요구사항
- 파일별 컬럼명을 먼저 점검하고 공통 기준 컬럼을 정리한다.
- 날짜 컬럼은 월 단위(YYYY-MM)로 통일한다.
- 업체명/브랜드명/카테고리명은 공백, 오탈자, 표기 불일치가 있으면 정제한다.
- 숫자 컬럼은 쉼표, 원화표시, 공백 등을 제거하여 수치형으로 변환한다.
- 주문 데이터는 분석용 fact table로 정리하고, 월 기준 summary table을 별도로 생성한다.
- 목표 데이터는 월 기준으로 매출 목표, 회원 목표를 join하여 KPI 계산이 가능하도록 한다.
- ERP 기준 실적 파일은 KPI 기준 master table로 사용한다.

# 계산 필드 정의
아래 계산 필드를 반드시 생성한다.

- 매출 목표 달성률 = 실제 매출 / 매출 목표
- 회원 목표 달성률 = 실제 회원 수 / 회원 목표
- ROAS = 매출 / 마케팅비
- 영업이익률 = 영업이익 / 매출 (가능하면 추가)
- 카테고리 매출 비중 = 카테고리 매출 / 전체 매출
- 업체 매출 비중 = 업체 매출 / 전체 매출

# 시각화 가이드
- KPI 카드는 숫자가 크게 보이도록 구성
- 증감률은 색상 또는 아이콘으로 직관적으로 표시
- 월별 추이는 라인차트 또는 막대차트 사용
- 순위형 데이터는 가로 막대차트 사용
- 상품/업체명은 너무 길면 줄바꿈 또는 말줄임 처리
- 경영진이 보기 쉬운 깔끔한 레이아웃으로 제작
- 불필요하게 복잡한 그래프보다 해석이 쉬운 차트를 우선 적용

# 최종 산출물 요구사항
1. 월 선택 필터가 적용된 인터랙티브 대시보드
2. 데이터 전처리 및 계산 로직 설명
3. 사용한 각 파일별 역할 설명
4. 시트/테이블 간 연결 구조 설명
5. 주요 KPI 정의서
6. 경영진 보고용으로 바로 활용 가능한 수준의 디자인

# 추가 요청
- 업로드한 raw 데이터의 컬럼명을 먼저 분석한 후, 어떤 파일에서 어떤 지표를 가져오는지 명확히 매핑해달라.
- 데이터 구조가 불완전할 경우, 추정하지 말고 누락 컬럼/추가 필요 항목을 먼저 정리해달라.
- 대시보드 제작 전, 먼저 아래 내용을 표로 정리해달라.
  1) 파일명
  2) 주요 컬럼
  3) 활용 목적
  4) 생성 가능한 KPI
  5) 다른 파일과의 연결 키
- 그 다음 실제 대시보드 설계안을 제안하고 제작한다.
